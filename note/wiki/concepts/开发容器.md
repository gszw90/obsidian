# 开发容器

用容器技术隔离、复现开发环境。避免「本机能跑，服务器不行」。

## mac 上的选项

- [[OrbStack]] — 轻量高性能（当前采用）
- Docker Desktop — 官方，较重
- Colima — 开源 CLI，最小化
- Podman Desktop — daemonless

## 何时用

- 多项目依赖隔离（不同 Node / Python / Go 版本）
- 复现生产环境
- 团队统一环境
