# OrbStack

macOS 原生的 Docker / Linux 虚拟机运行时。定位为 Docker Desktop 的轻量、高性能替代。

## 关键点

- **平台**：macOS（Apple Silicon + Intel）
- **作用**：运行 Docker 容器 + Linux 虚拟机，比 Docker Desktop 启动快、占用低
- **下载**：https://orbstack.dev/download

## 对比

| 维度 | OrbStack | Docker Desktop |
|---|---|---|
| 性能 | 快，资源占用低 | 较重 |
| Linux VM | 支持 | 有限 |
| 商用授权 | 个人/小团队免费 | 大企业需付费 |
