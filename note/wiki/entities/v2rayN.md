# v2rayN

v2rayN — Windows 平台 V2Ray / Xray 图形客户端，科学上网翻墙代理工具。基于 V2Ray 核心，支持 VMess / VLESS / Trojan / Shadowsocks 等协议。

## 关键点

- **作用**：翻墙代理客户端，订阅 / 配置节点
- **平台**：All（v2rayN 主指 Windows，另有 v2rayNG Android / v2rayU macOS）
- **仓库**：https://github.com/2dust/v2rayN/releases
- **核心**：可切换 V2Ray / Xray / sing-box
