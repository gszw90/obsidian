# gopeed

Gopeed — 开源高速下载器，支持 HTTP / BitTorrent / Magnet / ed2k。Go + Flutter 实现，全平台。

## 关键点

- **协议**：HTTP、BitTorrent、Magnet、ed2k
- **平台**：All（Windows / macOS / Linux / Android / iOS / Web）
- **仓库**：https://github.com/GopeedLab/gopeed/releases
- **特性**：多线程、轻量、跨端
