# 翻墙代理工具

科学上网代理客户端的统称。基于本地 HTTP / SOCKS 代理 + 远程节点，按规则路由流量绕过网络审查。两大生态：**V2Ray 系**（VMess/VLESS/Trojan/SS）与 **Clash 系**（mihomo / 规则订阅）。

## 选型维度

- **核心引擎**：V2Ray / Xray / sing-box（v2rayN 系）vs Clash / mihomo（clash-verge 系）
- **UI / 体验**：[[clash-verge]] 颜值优于 [[v2rayN]]
- **协议覆盖**：VLESS + Reality 抗封锁能力更强
- **订阅格式**：Clash YAML 订阅 vs V2Ray base64 订阅
