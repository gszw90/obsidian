# 浏览器扩展

Browser Extension — 运行于浏览器内的小程序，通过 WebExtensions API 改造浏览 / 网页行为。各浏览器（Edge / Chrome / Firefox / Safari）商店分发，多数跨平台兼容。

## 子类

- **脚本管理**：[[暴力猴]]（运行用户脚本）
- **翻译**：[[沉浸式翻译]]
- **广告拦截**：[[AdGuard]]
- **新标签页**：Infinity
- **开发辅助**：JSON 格式化工具

## 选型要点

- **同步**：是否支持账号云同步（暴力猴无，影响换机 / 重装体验）
- **平台**：Edge addons ≠ Chrome Web Store，跨平台需求需核对
- **开源 vs 闭源**：影响可信度与可审计性
