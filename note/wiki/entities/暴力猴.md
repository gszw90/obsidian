# 暴力猴

Violentmonkey — 开源浏览器用户脚本管理器（Userscript Manager），Tampermonkey 的轻量替代。支持 Userscripts.org / Greasy Fork 脚本。

## 关键点

- **作用**：安装并运行用户脚本（JS），改造网页行为 / 增强功能
- **平台**：Edge / Chrome / Firefox（本 vault 记录 Edge 链接）
- **同步**：无内置账号同步；新装浏览器需重装脚本

> [!warning] 同步缺口
> 源笔记标注：暴力猴无账号同步功能，每次新装浏览器都需重装相关插件。可配合云盘同步脚本目录缓解。

## 对比

| 维度 | 暴力猴 Violentmonkey | Tampermonkey | Greasemonkey |
|---|---|---|---|
| 开源 | 是 | 否（部分） | 是 |
| 轻量 | 较轻 | 较重 | 老牌，更新慢 |
| 同步 | 无 | 有（需配置） | 无 |
