# legado

legado（开源阅读 / 阅读 App）— Android 开源小说 / 漫画阅读器。通过导入书源（订阅源）实现免费阅读小说、漫画。书源质量决定体验。

> [!warning] 仓库状态：原仓库已清空
> 源笔记标注：原仓库作者因被告（法律纠纷）已清空仓库。本 vault 记录的链接为社区 fork：`HapeLee/legado-with-MD3`（Material Design 3 风格 fork）。

## 关键点

- **作用**：小说 / 漫画阅读，导入书源免费阅读
- **平台**：Android
- **本 fork**：https://github.com/HapeLee/legado-with-MD3/releases
- **关键**：书源质量决定内容可用性，优质书源很重要
