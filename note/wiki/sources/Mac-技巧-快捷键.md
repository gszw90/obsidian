# Mac 技巧 - 快捷键

> 来源：`mac/技巧.md`（个人 vault 原始笔记）

## 摘要

macOS 系统级快捷键速查。三条核心：

| 快捷键 | 作用 |
|---|---|
| `⌘ + I` | 查看文件信息（Get Info） |
| `⇧ + ⌘ + .` | 显示/隐藏隐藏文件夹 |
| `⇧` | 中英文输入法切换 |

## 详细

### 文件信息
`⌘ + I` → Finder 选中文件后查看「显示简介」窗口（大小、创建/修改时间、权限、扩展名等）。

### 隐藏文件
`⇧ + ⌘ + .` → 切换 Finder 中以 `.` 开头文件的可见性。macOS 默认隐藏 dotfiles。

### 输入法切换
`⇧`（单独按）→ 系统中文输入法下切换中/英模式。区别于系统级 `⌃ + 空格` 切换输入源。
