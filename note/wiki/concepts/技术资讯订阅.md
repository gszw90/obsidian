# 技术资讯订阅

定期更新的技术 / 开源 / 科技资讯来源。按更新频率和编辑模式分：**人工精选周刊** vs **爬虫聚合**。

## 来源对比

| 来源 | 模式 | 频率 | 见 |
|---|---|---|---|
| HelloGitHub | 人工精选 GitHub 项目 | 月刊式 | [[HelloGitHub]] |
| 科技爱好者周刊 | 阮一峰个人周刊 | 每周五 | [[科技爱好者周刊]] |
| infinitum | 爬虫新闻聚合 | 实时 | —（见 [[常用网址导航#资讯]]）|

## 选型要点

- **人工 vs 爬虫**：人工精选质量高但更新慢；爬虫聚合实时但需自筛
- **RSS 可订阅性**：周刊类多支持 RSS，便于聚合阅读
- **领域覆盖**：开源 / 综合 / 垂直
