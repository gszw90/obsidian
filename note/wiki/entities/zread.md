# zread

在线仓库解析服务。

## 关键点

- **网址**：https://zread.ai
- **作用**：输入 GitHub 仓库地址，自动解析产出：
  - 仓库概述
  - 安装方式
  - 目录结构
- **交互**：支持 AI 对话深入了解仓库、查询函数调用关系等

## 适用场景

- 快速熟悉新仓库（替代手翻 README）
- 搞清函数调用链 / 依赖关系
