# Mac 快捷键

macOS 系统快捷键约定。修饰键：`⌘`（Command）、`⇧`（Shift）、`⌃`（Control）、`⌥`（Option）。

## 常用

| 快捷键 | 作用 |
|---|---|
| `⌘ + I` | 文件显示简介 |
| `⇧ + ⌘ + .` | 显示隐藏文件 |
| `⇧` | 中英文输入法切换 |
| `⌘ + 空格` | Spotlight 搜索 |
